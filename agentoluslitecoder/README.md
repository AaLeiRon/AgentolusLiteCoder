## тЬи Features

| | Feature | Description |
|---|---|---|
| ЁЯза | **Local LLM Powered** | Runs on Ollama with any model тАФ GLM, Qwen, GPT-OSS, and more |
| ЁЯПЦя╕П | **Sandboxed Execution** | All code runs safely inside `~/agentolus_sandbox` |

## ЁЯЪА Quick Start

**Prerequisites:** Python 3.10+ and [Ollama](https://ollama.com) running locally

```bash
git clone https://github.com/AaLeiRon/agentoluslitecoder.git
cd agentoluslitecoder
pip install -e .
```

Then just type:

```bash
ald
```

That's it! ЁЯОЙ

> **Other modes:**
> - `alc "create a hello world script"` тАФ single prompt mode
> - `ald --sandbox /custom/path` тАФ custom sandbox location  
> - `ald --reset` тАФ clear conversation memory

## ЁЯЫбя╕П Security

> **Your system is always protected.** Every operation is sandboxed, sanitized, and logged.

| | Protection | How |
|---|---|---|
| ЁЯФТ | **Sandboxed** | All operations confined to `~/agentolus_sandbox` |
| ЁЯз╣ | **AST Sanitized** | Path injection attacks stripped at syntax tree level |
| ЁЯЪл | **No Shell Escape** | Shell metacharacters `& \| ; > <` blocked |
| ЁЯТ╛ | **Auto Backup** | Files backed up before any overwrite |
| ЁЯУЭ | **Audit Trail** | Every action logged to `execution.log` |
| ЁЯФС | **100% Offline** | No data ever leaves your machine |

## ЁЯМР HTTP API

> Connect AgentolusLiteCoder to **Telegram**, **Discord**, or any HTTP client

```bash
pip install fastapi uvicorn
python litecoder_api.py
# тЖТ Server running on http://localhost:8085
```

**Endpoints:**

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/chat` | Send a coding prompt |
| `POST` | `/api/reset` | Reset conversation memory |
| `GET` | `/api/status` | Sandbox & model info |
| `GET` | `/health` | Health check |

**Example:**

```bash
curl -X POST http://localhost:8085/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "create a hello world file"}'
```

```json
{
  "reply": "Created: hello.py",
  "code": "with open('hello.py', 'w') as f: ...",
  "cwd": "agentolus_sandbox"
}
```

## ЁЯПЧя╕П How It Works

```
  You тФАтФАтФАpromptтФАтФАтЖТ ЁЯза LLM (Ollama) тФАтФАтФАpython codeтФАтФАтЖТ ЁЯФз Processing Pipeline тФАтФАтФАsafe codeтФАтФАтЖТ ЁЯПЦя╕П Sandbox
                                                          тФВ
                                                     extract_code()
                                                     _sanitize_code()    тЖР AST sanitization
                                                     _repair_code()      тЖР auto-fix mistakes
                                                     _maybe_wrap_expr()  тЖР REPL-style output
```

All files live in `~/agentolus_sandbox/`:

